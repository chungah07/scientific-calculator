# 공학용 계산기

바이브코딩으로 만든 세련된 공학용 계산기입니다.

## 기능

- **기본 연산**: +, -, ×, ÷, %
- **공학 함수**: sin, cos, tan (DEG/RAD/GRAD 지원), 역삼각함수
- **로그/지수**: log, ln, eˣ, x², x³, xʸ
- **기타 함수**: √, ∛, |x|, ⌊x⌋, ⌈x⌉, 팩토리얼, +/−
- **상수**: π, e
- **메모리**: MC, MR, M+, M−, MS
- **키보드 입력 지원**
- **다크 테마 UI**

## 사용

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.
